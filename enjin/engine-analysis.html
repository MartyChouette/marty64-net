Key Pipeline Optimizations:
  • Scene Classification Gate: 2D-only scenes skip shadow passes entirely (4+ render passes saved)
  • Material Sort: Entities sorted by texture key for consecutive identical materials
  • Descriptor Caching: ~75-85% descriptor write reduction via last-bound tracking
  • Play Mode Skip: Prevents double-drawing (offscreen Game View + main swapchain)

Key Dependency Observations

  • VulkanContext is the foundation: Everything rendering-related flows through it
  • ECS::World is the central hub: Almost every system depends on World
  • Physics abstraction is well-isolated: IPhysicsBackend cleanly separates consumers from implementations
  • Scripting has broad reach: Bindings touch nearly every system
  • PlayMode is a system compositor: Wires together physics, scripting, VS, AI, audio
  • Graph editors are leaf nodes: Can be added/removed without architectural impact
  • RT pipeline is cleanly optional: Graceful fallback via RTCapabilities detection
