Basic Jump

A simple, fixed-height jump. Press button, go up, come down. The foundation of platformer movement.

The Feel

Basic jump is committed and consistent. Same height every time. The player knows exactly what they'll get. Compare to Variable Jump: less expressive, but more predictable. Some videogames want that predictability.

Exposed Variables

VariableTypeDefaultWhat it does

jumpForcefloat12.0Initial upward velocity gravityfloat40.0Downward acceleration fallGravityMultiplierfloat1.5Extra gravity when falling groundCheckDistancefloat0.1How far to check for ground

Tuning Guide

For a floaty feel:

- Lower gravity (20-30) - Lower fallGravityMultiplier (1.0-1.2) - Higher jumpForce Reference: Kirby, early Mario

For a snappy feel:

- Higher gravity (50-70) - Higher fallGravityMultiplier (2.0-3.0) - Moderate jumpForce Reference: Super Meat Boy, modern precision platformers

For a weighty feel:

- High gravity - Low jumpForce - Minimal fallGravityMultiplier (1.0) Reference: Castlevania, simulation-leaning games

How It Works

On Jump Press (if grounded):
  - Apply jumpForce upward immediately
Every Frame:
  - If rising: apply gravity
  - If falling: apply gravity * fallGravityMultiplier
  - Check for ground contact

The fallGravityMultiplier is the key insight: making descent faster than ascent creates a satisfying arc without the floaty falling that pure physics produces.

Common Issues

"It feels floaty" → Increase fallGravityMultiplier. Most good-feeling jumps have faster descent than ascent. "It's too short/tall" → Adjust jumpForce and gravity together. Higher force + higher gravity = same height, different speed. "It feels like I'm on the moon" → Gravity is too low. Real platformers use 2-4x Earth gravity. "I can jump in the air" → Ground check isn't working. Check groundCheckDistance and your collision layers.

Combine With

- Coyote Time — grace period makes basic jump more forgiving - Input Buffer — catches jumps pressed slightly before landing - Squash Stretch — visual feedback for the jump - Landing Impact — feedback when you land

Heritage Notes

The basic jump predates videogames — it's how jumping works in reality. But videogame jumps have always been exaggerated: - Donkey Kong (1981): Fixed arc jump - Super Mario Bros. (1985): Introduced variable jump, but basic enemies still use fixed jump - Modern games: Variable is default, but basic persists in certain contexts
